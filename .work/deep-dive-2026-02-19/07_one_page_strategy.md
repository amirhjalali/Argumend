# Argumend One-Page Strategy

## Strategy in One Sentence
Win attention with shareable social hooks, convert that attention through a format ladder, and retain trust with transparent, reciprocal deliberation.

## North Star
Argumend becomes the default destination where people can go from a 30-second hook to evidence-level understanding without losing nuance.

## The 3-Engine Model

### 1) Attention Engine (Acquisition)
- Primary channel: social hook content.
- Core formats: 30-60s video, carousels, short clips, creator remixes.
- Hook pattern: contradiction -> crux -> uncertainty -> deep-link CTA.

### 2) Comprehension Engine (Conversion)
- Format ladder: 30s recap -> 2m explainer -> 5m brief -> full interactive map.
- Every social artifact maps to exact topic/pillar/crux node.
- Goal: make depth feel easy, not intimidating.

### 3) Trust Engine (Retention)
- Reciprocal steelman + claim-type clarity (fact/norm/sincerity).
- Confidence ledger and evidence challenge workflow.
- Outcome: users return because the process is fair and auditable.

## Core Flywheel
Social Hook -> Click -> 30s/2m/5m Summary -> Deep Map Session -> Debate/Participation -> Shareable Output -> More Hooks

## What We Build First (Next 6 Weeks)
1. 30s/2m/5m summary modes on topic pages.
2. Topic-to-carousel generator.
3. Topic-to-short-video script generator.
4. Exact-node deep links for every generated asset.
5. Funnel analytics: hook hold -> completion -> click-through -> deep session -> return.

## What We Do Later (After Baseline Traffic)
1. Full podcast network production.
2. Calibration reputation system.
3. Advanced governance and arbitration workflows.
4. Complex community incentive layers.

## Non-Negotiable Product Rule
No feature ships unless it clearly improves one of these:
- acquisition,
- depth conversion,
- trust/retention.

## Success Metrics
- Acquisition: hook retention, share rate, profile clicks.
- Conversion: social-to-site CTR, summary-to-deep transition rate.
- Retention: repeat deep sessions, returning weekly users.
- Trust: source click-through, evidence challenge participation, confidence-update acceptance.

## Weekly Operating Cadence
- 5 hook videos
- 2 carousels
- 1 deep breakdown
- 1 confidence-change digest
- 1 review meeting focused only on funnel metrics and topic winners

## Strategic Filter
If content is engaging but not deep-linkable, it is brand media, not product growth.
If content is rigorous but not shareable, it is documentation, not acquisition.
Argumend must do both in one loop.
